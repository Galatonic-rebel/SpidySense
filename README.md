# 🕷️ SpidySense

SpidySense is a real-time computer vision project that uses hand tracking and gesture recognition to simulate Spider-Man-style web shooting through a webcam.

Using OpenCV and MediaPipe, the system detects hand landmarks, recognizes the Spider-Man hand gesture, and generates animated web effects based on hand movement and orientation. The project also includes sound effects and gesture-based interaction through a virtual reset button.

## Features

* Real-time hand tracking
* Spider-Man gesture recognition
* Motion-based web shooting
* Animated web effects
* Sound effects
* Virtual reset button

## Tech Stack

* Python
* OpenCV
* MediaPipe
* NumPy
* Pygame

## Run

```bash
pip install -r requirements.txt
python spiderman_web.py
```
